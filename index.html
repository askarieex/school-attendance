Key Architecture Decisions

Dashboard Architecture

Decision: TWO Separate Dashboard Applications

  • Super Admin Dashboard (admin.attendanceapp.com) - Internal management tool, dark theme
  • School Admin Dashboard (app.attendanceapp.com) - Client portal, light theme
Multi-Tenancy Pattern

Every table has school_id. Every query MUST filter by school_id for complete data isolation.

Real-Time Updates

Dashboard polls API every 30 seconds. SMS notifications sent within 5 seconds of check-in.
